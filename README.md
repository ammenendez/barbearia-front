# Barbearia Front

Frontend React para a agenda de serviços da barbearia.

## Stack

- React
- TypeScript
- Vite
- React Router DOM
- Axios
- React Hook Form
- Zod
- Tailwind CSS

## Setup

```bash
npm install
cp .env.example .env
npm run dev
```

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run lint`

## Estrutura

- `/dashboard` para visão geral
- `/servicos` para o módulo principal

## Observações

O projeto já vem preparado para consumir a API em `VITE_API_BASE_URL`.
